Install in 4 steps
1
Open your Google Sheet → Extensions → Apps Script
2
Paste code.js into Code.gs. Click + next to Files → New script → name it Sidebar → paste Sidebar.gs.
3
Project Settings → check Show appsscript.json → paste appsscript.json into it. Save all files.
4
Select function setup → Run → authorise when prompted. Creates Config sheet, all Gmail labels (including the bare rally parent), and sets the trigger.
To activate the Gmail sidebar: Deploy → Test deployments → Install. The scoring panel will appear automatically on any rally submission email.
