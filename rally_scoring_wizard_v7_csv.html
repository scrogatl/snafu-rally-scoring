3 steps to finish setup
1
Import config.csv into your spreadsheet
File → Import → Upload → select config.csv
Import location: Insert new sheet(s) — then rename the new tab to exactly Config
2
Paste the script files into Apps Script
Open your Sheet → Extensions → Apps Script
Paste code.js into Code.gs, add a new file named Sidebar and paste Sidebar.gs
Enable the manifest, paste appsscript.json, and save all files
3
Run setup()
Select function setup → Run → authorise when prompted
This reads your Config sheet, creates all Gmail labels, builds rider score sheets, and sets the trigger
To reconfigure: edit values directly in the Config sheet, then re-run setup(). No need to use this wizard again.
